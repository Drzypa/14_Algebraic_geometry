analytic algebraic function


Let k be a field, and let k⁢{x1,…,xn} be the ring of convergent power series in n variables. An element in this ring can be thought of as a function defined in a neighbourhood of the origin in kn to k. The most common cases for k are ℂ or ℝ, where the convergence is with respect to the standard euclidean metric. These definitions can also be generalized to other fields.

Definition.

A function f∈k⁢{x1,…,xn} is said to be k-analytic algebraic if there exists a nontrivial polynomial p∈k⁢[x1,…,xn,y] such that p⁢(x,f⁢(x))≡0 for all x in a neighbourhood of the origin in kn. If k=ℂ then f is said to be holomorphic algebraic and if k=ℝ then f is said to be real-analytic algebraic or a Nash function.

The same definition applies near any other point other then the origin by just translation.

Definition.

A mapping f:U⊂kn→km where U is a neighbourhood of the origin is said to be k-analytic algebraic if each component function is analytic algebraic.
