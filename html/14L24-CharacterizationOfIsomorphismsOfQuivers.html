characterization of isomorphisms of quivers


Let Q=(Q0,Q1,s,t) and Q′=(Q0′,Q1′,s′,t′) be quivers. Recall, that a morphism F:Q→Q′ is an isomorphism if and only if there is a morphism G:Q′→Q such that F⁢G=Id⁢(Q′) and G⁢F=Id⁢(Q), where

Id⁢(Q):Q→Q

is given by Id⁢(Q)=(Id⁢(Q)0,Id⁢(Q)1), where both Id⁢(Q)0 and Id⁢(Q)1 are the identities on Q0, Q1 respectively.

Proposition. A morphism of quivers F:Q→Q′ is an isomorphism if and only if both F0 and F1 are bijctions.

Proof. ,,⇒” It follows from the definition of isomorphism that F0⁢G0=Id⁢(Q′)0 and G0⁢F0=Id⁢(Q)0 for some G0:Q0′→Q0. Thus F0 is a bijection. The same argument is valid for F1.

,,⇐” Assume that both F0 and F1 are bijections and define G:Q0′→Q0 and H:Q1′→Q1 by

G=F0-1,H=F1-1.

Obviously (G,H) is ,,the inverse” of F in the sense, that the equalites for compositions hold. What is remain to prove is that (G,H) is a morphism of quivers. Let α∈Q1′. Then there exists an arrow β∈Q1 such that

F1⁢(β)=α.

Thus

H⁢(α)=β.

Since F is a morphism of quivers, then

s′⁢(α)=s′⁢(F1⁢(β))=F0⁢(s⁢(β)),

which implies that

G⁢(s′⁢(α))=s⁢(β)=s⁢(H⁢(α)).

The same arguments hold for the target function t, which completes the proof. □
