Selmer group


Given an elliptic curve E we can define two very interesting and important groups, the Selmer group and the Tate-Shafarevich group, which together provide a measure of the failure of the Hasse principle for elliptic curves, by measuring whether the curve is everywhere locally soluble. Here we present the construction of these groups.

Let E,E′ be elliptic curves defined over ℚ and let ℚ¯ be an algebraic closure of ℚ. Let ϕ:E→E′ be an non-constant isogeny (for example, we can let E=E′ and think of ϕ as being the “multiplication by n” map, [n]:E→E). The following standard result asserts that ϕ is surjective over ℚ¯:

Theorem 1.

Let C1,C2 be curves defined over an algebraically closed field K and let

ψ:C1→C2

be a morphism (or algebraic map) of curves. Then ψ is either constant or surjective.

Proof.

See [4], Chapter II.6.8. ∎

Since ϕ:E⁢(ℚ¯)→E′⁢(ℚ¯) is non-constant, it must be surjective and we obtain the following exact sequence:

0→E⁢(ℚ¯)⁢[ϕ]→E⁢(ℚ¯)→E′⁢(ℚ¯)→0  (1)

where E⁢(ℚ¯)⁢[ϕ]=Ker⁡ϕ. Let G=Gal⁡(ℚ¯/ℚ), the absolute Galois group of ℚ, and consider the it⁢h-cohomology group Hi⁢(G,E⁢(ℚ¯)) (we abbreviate by Hi⁢(G,E)). Using equation (1) we obtain the following long exact sequence (see Proposition 1 in group cohomology):

0→H0⁢(G,E⁢(ℚ¯)⁢[ϕ])→H0⁢(G,E)→H0⁢(G,E′)→H1⁢(G,E⁢(ℚ¯)⁢[ϕ])→H1⁢(G,E)→H1⁢(G,E′)  (2)

Note that

H0⁢(G,E⁢(ℚ¯)⁢[ϕ])=(E⁢(ℚ¯)⁢[ϕ])G=E⁢(ℚ)⁢[ϕ]

and similarly

H0⁢(G,E)=E⁢(ℚ),H0⁢(G,E′)=E′⁢(ℚ)

From (2) we can obtain an exact sequence:

0→E′⁢(ℚ)/ϕ⁢(E⁢(ℚ))→H1⁢(G,E⁢(ℚ¯)⁢[ϕ])→H1⁢(G,E)⁢[ϕ]→0

We could repeat the same procedure but this time for E,E′ defined over ℚp,for some prime number p, and obtain a similar exact sequence but with coefficients in ℚp which relates to the original in the following commutative diagram (here Gp=Gal⁡(ℚp¯/ℚp)):

0→E′⁢(ℚ)/ϕ⁢(E⁢(ℚ))→ H1⁢(G,E⁢(ℚ¯)⁢[ϕ]) →H1⁢(G,E)⁢[ϕ]→0
↓ ↓    ↓
0→E′⁢(ℚp)/ϕ⁢(E⁢(ℚp))→ H1⁢(Gp,E⁢(ℚp¯)⁢[ϕ]) →H1⁢(Gp,E)⁢[ϕ]→0

The goal here is to find a finite group containing E′⁢(ℚ)/ϕ⁢(E⁢(ℚ)). Unfortunately H1⁢(G,E⁢(ℚ¯)⁢[ϕ]) is not necessarily finite. With this purpose in mind, we define the ϕ-Selmer group:

Sϕ⁢(E/ℚ)=Ker⁡(H1⁢(G,E⁢(ℚ¯)⁢[ϕ])→∏pH1⁢(Gp,E))

Equivalently, the ϕ-Selmer group is the set of elements γ of H1⁢(G,E⁢(ℚ¯)⁢[ϕ]) whose image γp in H1⁢(Gp,E⁢(ℚ𝕡¯)⁢[ϕ]) comes from some element in E⁢(ℚp).

Finally, by imitation of the definition of the Selmer group, we define the Tate-Shafarevich group:

T⁢S⁢(E/ℚ)=Ker⁡(H1⁢(G,E)→∏pH1⁢(Gp,E))

The Tate-Shafarevich group is precisely the group that measures the Hasse principle in the elliptic curve E. It is unknown if this group is finite.
